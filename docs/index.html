Introduction

This project provides sheet music for Henri Eccles (1670-1742) Sonata in G Minor, along with piano accompaniment arranged by Joseph Salmon (1864-1943). The source for this transcription is J. Salmon's 1914 edition of "Oeuvre D'Auteurs Anciens", Société Anonyme des Éditions Ricordi. The original work is in public domain, and should be available on the Internet via various providers.

The transcription source is provided in LilyPond notation. The PDF output for these are

  • Eccles-G-Minor-All
  • Eccles-G-Minor-Violin
